# 프로젝트 구동

1. node.js & node version manager & npm 설치

- [설치가이드 블로그](https://seunghyun90.tistory.com/52)

2. 프로젝트 루트 경로에서 다음 커멘드 실행:

```
  npm install
```

3. 설치완료 후에 다음 커멘드 실행:

```
  npm start
```
